## 🚀 Features

- ✅ **No Watermark** — Download TikTok videos without the TikTok logo
- ✅ **HD Quality** — Up to 1080p MP4 video downloads
- ✅ **MP3 Audio** — Extract audio from any TikTok video
- ✅ **Photo Mode** — Download TikTok Photo Mode as JPG/PNG
- ✅ **No App Required** — Works entirely in your browser
- ✅ **All Devices** — iPhone, Android, Windows, Mac, Linux
- ✅ **No Registration** — Completely free, no signup needed
- ✅ **Multi-Language** — 20 languages supported

## 🛠️ Tech Stack

- **Frontend**: Next.js 15, React 19, TypeScript
- **Styling**: Tailwind CSS, Radix UI
- **Deployment**: Vercel + Cloudflare CDN
- **SEO**: Multi-language sitemap, JSON-LD Schema, hreflang

## 📋 How to Use

1. Open TikTok and copy the video link
2. Paste the link at [saveik.com](https://www.saveik.com)
3. Click "Download"
4. Choose MP4 (video) or MP3 (audio)
5. Save to your device
